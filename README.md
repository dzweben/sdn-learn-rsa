# sdn-learn-rsa

Personal learning and reference materials related to representational similarity
analysis (RSA), including notes, papers, slides, and example code.

## Structure

- Learn/
  - Background/ — background emails, PDFs, and reference docs
  - Papers_Presentation/ — papers and presentation files
  - RSA_Coding/ — code and notes for RSA workflows
  - RSA papers/ — curated RSA papers
  - Social Learning/ — related learning materials
  - Source-Githubs/ — third-party example repositories (see licensing below)
- Meetings/ — meeting notes
- SA_Review/ — related review documents
