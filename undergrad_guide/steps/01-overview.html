Project Overview

Big picture: what LEARN measures, what RSA does, and how this guide moves from theory to execution.

What You Are Doing

This project analyzes how social learning is represented in the brain. You will learn the dataset, the workflow, and the scripts that generate beta maps for RSA.

We will build intuition first, then move into actual command-line execution in later steps.

Why It Matters

RSA asks how similar neural patterns are across conditions. In this project, those conditions are social interactions across runs and peers.

The end goal is to test how well brain representations mirror the true peer structure, and how this differs with social anxiety.
